# Assignment 1 - Principles of Data Science

## Dataset: StudentsPerformance
- Source: Local copy of the public `StudentsPerformance.csv` dataset (originally from Kaggle).
- Records: 1000 students, 8 columns capturing demographics and test scores.

### Columns
- `gender`: Student's gender (`female` or `male`).
- `race/ethnicity`: Student's racial/ethnic group label.
- `parental level of education`: Highest level of education attained by either parent.
- `lunch`: Type of lunch received (`standard` or `free/reduced`).
- `test preparation course`: Whether the student completed a test prep course.
- `math score`: Math exam score (0-100).
- `reading score`: Reading exam score (0-100).
- `writing score`: Writing exam score (0-100).
